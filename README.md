QMon is a redis monitor 

# Installation
```shell
pip3 install qmon
```

# Usage:

```shell
> export HIP_TOKEN=MyHipChatToken
> qmon --host localhost --port 6379 --queue-name a_key_in_redis --room RoomInHipChat
[qmon.monitor] [2017-05-12 12:36:45,248] INFO : Connecting to hipchat
```
